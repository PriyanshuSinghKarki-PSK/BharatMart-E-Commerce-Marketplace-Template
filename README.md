# BharatMart – E-Commerce Marketplace Template

An open-source, feature-rich starting point for modern online storefronts. BharatMart is designed with a mobile-first approach, offering a seamless shopping experience with built-in state management for carts and wishlists, dynamic filtering, and a persistent dark mode.

## 🚀 Features

* **Dynamic Shopping Cart:** Add, remove, and adjust quantities with real-time subtotal calculations and a slide-out cart panel.
* **Product Wishlist:** Save favorite items to a local storage-backed wishlist.
* **Dark/Light Mode:** Toggle between themes seamlessly, with user preference saved via `localStorage`.
* **Quick View Modal:** Inspect product details, ratings, and descriptions without leaving the main page.
* **Category Filtering & Search:** Easily navigate through products using the search bar or category filter buttons.
* **Engaging UI/UX:** Features a countdown timer for flash sales, a scrolling brands ticker, scroll-reveal animations, and toast notifications for user actions.
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom properties (CSS variables) for theming, CSS Grid/Flexbox for layouts, and smooth keyframe animations.
* **Vanilla JavaScript (ES6+):** No external frameworks or libraries. Handles state management, DOM manipulation, and UI logic.
